Methods

__nextLeft

对于两个没有交叉的基因，不做任何附加处理。对于两个有相交部分的基因，则前一个基因会缩短以防止重叠，假若某一个基因完全的包裹另外一个基因，则也将不会做任何处理

Parameter Name Remarks
NextLeft 这个是基因组上面的位置，不是画图的位置
RefPoint 参数里面的@”P:System.Drawing.Point.X”参数就是当前的这个基因在绘图的时候的@”P:LANS.SystemsBiology.AnalysisTools.DataVisualization.MapModelCommon.Left”在图上面的位置
